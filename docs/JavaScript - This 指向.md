## This 指向

this 指向是 JavaScript 中一个非常重要的概念，它决定了函数内部的 this 指向，也就是函数执行时的上下

文。this 指向的值取决于函数的调用方式，它的值在函数被调用时才会被确定，而不是在函数被定义时确定。
