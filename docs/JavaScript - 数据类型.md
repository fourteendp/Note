## 八种基本数据类型

- Boolean
- Null
- Undefinde
- Number
- String
- Symbol
- BigInt
- Object

## Object 中包含那几中常见类型

Data、Function、Array 等都是 Object 类型

## 基本类型和引用类型

基本类型：非 Object 的类型都是基本类型

引用类型：Object 类型都是引用类型
