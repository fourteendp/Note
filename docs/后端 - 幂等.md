| Method | 是否会修改资源状态 | 是否幂等 | 是否可被缓存 | 含义          |
| ------ | --------- | ---- | ------ | ----------- |
| GET    | NO        | YES  | YES    | 查询          |
| PUT    | YES       | YES  | NO     | 全量更新一个资源    |
| PATCH  | YES       | YES  | NO     | 更新一个资源的指定字段 |
| DELETE | YES       | YES  | NO     | 删除一个资源      |
| POST   | YES       | NO   | NO     | 新增一个资源      |
