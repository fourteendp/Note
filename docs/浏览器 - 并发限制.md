浏览器的并发限制是针对单个域名的，已 chrome 为例限制一个域名支持最大并发数为 6,这其实是 HTTP1.1 协议的限制，HTTP1.1 一个域名只有一个 TCP 通道所以同一时刻只能处理一个 HTTP 请求，并且必须按照顺序。

**解决办法**

1. 同时采用多个域名请求，限制每个域名超过最大并发数，则替换到下一个域名请求，通过 DNS 解析到同一个 IP 地址
2. HTTP2 多路复用
